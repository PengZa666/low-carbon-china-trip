# 美团单车「低碳中国行」地图探索版

基于 PRD 实现的轻量级 Web 版：用骑行「油量」驱动虚拟小人在地图上移动，点亮城市并领取城市专属权益。

## 功能概览

- **背景层**：极简中国轮廓图，浅灰/淡蓝风格
- **城市节点**：18 个城市，未解锁为灰色，解锁后为美团黄并带光效
- **路线**：城市间虚线连接，小人沿虚线移动
- **动态层**：2D 骑车小人，沿预设路线滑行动画
- **操作层**：
  - 顶部：当前油量进度条、已解锁城市数、总电量
  - 中部：当前城市地标剪影与名称
  - 底部：「立即出发」按钮（每次 -10 油量）
- **到达新城市**：毛玻璃欢迎卡片 + 里程展示
- **领取纪念品**：城市特色优惠券弹窗

## 使用方式

1. 用浏览器直接打开 `index.html`，或本地起一个静态服务：
   ```bash
   npx serve .
   ```
2. 点击「立即出发」消耗 10 油量，小人从当前城市沿虚线移动到下一个未解锁城市。
3. 到达后节点变黄并弹出欢迎卡片，点击「领取纪念品」获得该城市专属券。

## 文件说明

- `index.html`：页面结构、地图 SVG、控制面板与弹窗
- `styles.css`：样式（含能量条、节点、弹窗、按钮等）
- `data.js`：中国轮廓路径、城市列表、路线与常量
- `app.js`：状态、渲染、移动动画、弹窗与领奖逻辑

## 技术说明

- 纯前端，无构建步骤，无后端
- 中国轮廓为极简 SVG 路径，可替换为更精细的轮廓数据
- 城市与路线在 `data.js` 中配置，便于扩展
