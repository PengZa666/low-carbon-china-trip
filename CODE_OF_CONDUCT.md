# 行为准则 (Code of Conduct)

## 我们的承诺

为了营造一个开放、友好、包容的社区环境，作为本项目（**low-carbon-china-trip**）的维护者和贡献者，我们承诺：无论参与者的背景、性别认同、性取向、能力、民族、外貌、种族、年龄、宗教或信仰如何，每个人在参与过程中均不受骚扰。

## 我们的标准

为了创造积极的社区环境，我们鼓励以下行为：

* 使用包容、友善的语言。
* 尊重不同的观点和经验。
* 优雅地接受建设性的批评。
* 关注对社区最有利的事情。
* 对其他社区成员表现出同理心。

我们不接受以下行为：

* 性暗示的语言、图像，以及任何形式的性骚扰。
* 侮辱、歧视、贬低性的言论，或人身攻击。
* 公开或私下的骚扰。
* 未经明确许可，发布他人的私人信息（如家庭住址或电子邮箱）。
* 其他在职业环境中被合理认为是不当的行为。

## 我们的责任

项目维护者有责任澄清并执行可接受行为的标准。对于任何表现出不当、威胁、冒犯或有害行为的参与者，维护者有权采取适当且公正的纠正措施，包括但不限于删除评论、禁止贡献，甚至取消其参与资格。

## 适用范围

本行为准则适用于项目的所有相关空间（包括 GitHub Issue、Pull Request 等），也适用于代表本项目在公共领域进行交流的场景。
